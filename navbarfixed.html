РасчетМеталл

Bootstrap - интуитивно простой и в тоже время мощный интерфейсный _Фрейморк_, повышающий скорость и облегчающий разработку web-приложений.

Learn more

Создан фанатами для фанатов.

Построенный на основе Twitter’а от @mdo и @fat, Bootstrap использует LESS CSS, компилирован с помощью Node, и управляется GitHub, помогая фанатам делать уникальные вещи в Интернете.

Сделан для всех.

Bootstrap совместим не только со всеми существующими браузерами (даже IE7!), но также используется в планшетах и смартфонах, благодаря наличию адаптивной CSS.

Сплошные преимущества.

12-колонная адаптивная сетка дизайна, десятки компонентов, плагины "JavaScript", оформление, создание разных форм, и даже основанный на интернет-технологии Customizer позволяют настроить Bootstrap индивидуально для себя.

Самый популярный инструмент для разработки адаптивных и мобильных web-проектов.

Bootstrap использует самые современные технологии CSS и HTML.

Learn more

Все еще ищете причины чтобы влюбиться в Bootstrap?

Препроцессоры

В дополнение к CSS, Bootstrap включает в себя поддержку двух самых популярных CSS препроцессоров, Less и "Sass".

Один код для всех устройств.

Bootstrap легко и эффективно масштабирует ваш проект с одной базой кода, от телефонов и планшетов до настольных компьютеров.

Документация

С Bootstrap, вы получаете обширную и прекрасную документации с сотнями живых примеров, фрагментов кода, и многое другое.

Создать Landing Page за 30 минут с Bootstrap.

Статья предназначена для тех, кто решил самостоятельно освоить и создать одностраничный сайт типа Landing Page.

На сайте http://bootstrap-3.ru/bootstraptheme.php скачать любой шаблон.

Принципиальное отличие html-странички для Landing Page состоит в:

1) добавлении в строку навигации аргумента class="affix", пример добавления в меню ....(ul) class="nav navbar-nav" class="affix"....

2) Добавление символа «#» в строку меню, пример ...(li) href="#forms">Forms (/li)....

Заключение любого блока, для которого есть ссылка в меню ( или любая навигационная ссылка) в следующей последовательности открывающий тег «section» сам блок Landing Page информации и закрытием блока тегом «/section»

Пример:

(section id="forms")

(div) class="container clearfix"

(div) class="row clearfix"

(div) class="col-lg-12"

(div) class="page-header"

(id="forms")готовые формы Forms

Любое содержимое контента Landing Page……

(/div)

(/div)

(/div)

(/div)

(/section)

3) Создайте множество блоков по этому типу, согласно наличии навигационных ссылок и вуаля Ваш Landing Page, Ваша страничка работает.

4) Все компоненты для наполнения можно взять там же http://bootstrap-3.ru/components.php.

5) Для примера, сохраните мою страницу Landing Page полностью, и по аналогу наполните свой сайт… поменяйте урлы, домены, заголовки.

6) Удачи Вам!!!
